# SureWinVIP

## 系统环境
1. **操作系统**  
   支持Win 8、win 10、win 11及以上64位版本。目前不支持mac和linux系统。  

2. **显示**  
   为保证显示效果，屏幕分辨率最小应是1920*1080。推荐2k屏幕显示。  

3. **软件环境**   
   必须安装windows access driver（2016或以上的64位版本）  
   访问驱动下载地址：https://www.microsoft.com/en-us/download/details.aspx?id=54920
   （如果下载地址失效，请自行查找后下载）
